asyncLoader

by Adrian Soluch

simple asynchronous script and stylesheet loading with ( optional ) callback

Load all files ( js & css ) asynchronously, simply because it's faster & makes your site faster.
Works in all major browsers :

  • Chrome
  • IE6+
  • Firefox
  • Safari
  • Opera

This example page heavy utilizes Twitter Bootstrap & Font Awesome 3.0

This stuff s being fadeIn() via jQuery by callback after jQuery was loaded asynchronously
Usage :

The only script you need to load is asyncLoader.js : 

                
        <script src="asyncLoader.js"></script>

Then load everything else you need & initialise via callback ( which is, by the way, optional ). 

                
        asyncLoader(
            [
                'css/bootstrap.min.css',
                'css/bootstrap-responsive.min.css',
                'css/font-awesome.css',
                'http://yandex.st/highlightjs/7.3/styles/pojoaque.min.css',
                '//ajax.googleapis.com/ajax/libs/jquery/1.9.0/jquery.min.js',
                '//ajax.googleapis.com/ajax/libs/jqueryui/1.10.0/jquery-ui.min.js',
                'http://yandex.st/highlightjs/7.3/highlight.min.js'
            ],
            {
                'callback':function() {
                    $('body').fadeIn( 800 );
                    $('#test').delay(1000).fadeIn();
                    // Syntax Highlighting
                    $('pre code').each(function(i, e) {hljs.highlightBlock(e)});
                }
            }
        );
